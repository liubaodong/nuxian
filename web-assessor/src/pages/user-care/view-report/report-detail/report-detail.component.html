本报告在PMP2T框架下全面评估（ 未来的 ）管理者的能力、技能、风格、动机，深度
评估内在的真实动力、性格模式，辅助人力资源部门、用人部门和主管领导等人事决策者全
面、深入的了解候选人，为进一步的人事决策提供客观量化分析以及纵向横向对比。

第一部分 报告总览

呈现受测人在各个模块的测评结果，可以通过此部分的内容了解受测人的全貌

第二部分 效率素质报告

呈现受测人言语理解、逻辑判断、数量关系、资料分析和思维策略等能力，并且将受测
人效率素质对团队效能的贡献进行分析

第三部分 动力素质报告

呈现受测人动机和社会愿望对受测人产生的影响程度，并描述该因素在团队中及大数据
标杆人才中的分布情况

第四部分 风格素质报告

呈现受测人人格特征及认知行为风格，对受测人的人格行为进行分析并提出策略

第五部分 适岗性报告

呈现受测人对组织贡献、领导素质、潜在缺陷、偏好工作环境以及发展建议
