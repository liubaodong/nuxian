在阅读本报告前，请您了解以下原则：

第一，人才测评是基于受测人的能力、个性和认知行为风格等指标进行测量的，没有对 受测人的专业知识、经验进行评估，如果用于重要的人事决策请联系沐贤智库联席专家。

第二，报告呈现的分数是依据原始分数与常模对照群体进行比较后的标准分数，如果对 照群体不同，即分数参照的标准不同，同样的原始分数转化成的标准分数也有所差异，对分 数的解释相应的也会发生变化。

第三，本报告中涉及的维度较多，您可以根据实际需要选择重点维度深入考察和分析。

阅读报告思路

第一，通过整体匹配度考察候选人的胜任度，可以在多个候选人之间进行横向比较。

第二，对每一项岗位胜任力进行分析，考察候选人与目标岗位要求之间的符合程度及在 工作中的典型表现。

第三，深入分析需要培训的胜任力项，结合其对岗位的重要性、个体的自我认知等，确 定需要培训的胜任力，并提供相应的培训发展课程和学习资源，辅助管理者快速成长。

术语解释

社会称许性反应（Social Desirability Responding ，SDR），指受测者倾向于按照现行为社会规范和准则过度积极自我描述， 而非根据自身真实情况选择答案的倾向。这种反应会使测评结果产生偏差。

为了保证心理测评的有效性，心理学家在测评中使用社会称许性得分衡量在测评过程中受测者掩饰真实自我的可能性。 社会赞许性得分越高，表示受测者获得这会赞赏和接受的心理需求越强，心理防卫程度越高，测验结果反应受测人真实情况的可能性越小。

效率素质报告：管理能力倾向

管理能力倾向测验考察的是受测人日常工作中最常用的基本能力，体现从事具体工作的效率。这里所指的“能力”既有智力因素也有经验、 技能因素：智力因素很大程度上预测了人们的潜力，经过长期的努力才可以有部分改变的； 而经验、技能因素可以通过一段相对较短时间的训练获得迅速提升。

管理能力倾向测验的结果只代表受测者的所测能力水平，与工作绩效无直接关系。通常来说，能力水平高不意味着一定能将工作做好， 而能力水平低则意味着将工作做好需更多努力和提高。

管理能力总体情况

本测验综合反应了受测人管理能力的两个方面：一为与管理能力相关的一般智力水平，一为综合应用能力水平。 与管理能力相关的一般智力水平主要是指受测者的语言能力、逻辑思维能力和数量关系能力， 而综合应用能力是指受测者综合应用上述三种能力分析资料和解决问题的能力。

管理能力倾向测验结果显示，受测人企业管理能力测验整体水平较高（常模百分位为75%，等级为2级）。 说明在工作实践中，受测者思维反应敏捷，分析判断问逻辑清楚，学习领悟能力良好，具有良好的管理能力基础。 受测者的这些素质优势，有助于其管理能力获得良好的发展和表现，其管理能力基础符合现代企业合格管理人员的基本要求。
